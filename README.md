🌦️ Historical Weather Dashboard

A responsive, interactive weather application that retrieves and analyzes historical data from the Open-Meteo API. This dashboard provides users with visual trends and detailed statistical breakdowns for temperature, wind speed, and precipitation probability.

🚀 Live Demo
[Historical Weather Dashboard](https://25-bowd-18.course.tamk.cloud/weather_dashboard/index.html).

✨ Features
1. Multi-View Navigation

      - Temperature View: Hourly tracking of the last 20 readings with history up to 1 month.
    
      - Windspeed View: Visualization of wind velocity in m/s.
    
      - Precipitation View: Monitoring the probability of precipitation over time.

2. Interactive Visualization
   
      - Dynamic Charting: Integrated Chart.js to render line graphs that update in real-time based on user selection.
    
      - Customizable Timespans: Users can toggle between the latest 20 readings, 24 hours, 48 hours, 1 week, and 1 month.
    
      - Unified UI: A sleek "DodgerBlue" theme applied to all charts for a professional, cohesive look.

3. Advanced Statistical Analysis
   
      - Calculates and displays eight key statistical indicators for every selected timespan:
    
      - Central Tendency: Mean (Average), Median, and Mode.
    
      - Variability: Range, Standard Deviation, and Variance.
    
      - Extremes: Minimum and Maximum values.

4. Responsive Design
   
      - Bootstrap 5: Utilized for a mobile-friendly grid layout and navigation components.
    
      - Sticky Headers: Historical data tables feature sticky headers to maintain context while scrolling.

🛠️ Technology Stack
      - Frontend: HTML5, CSS3, Vanilla JavaScript (ES6+).

      - Framework: Bootstrap 5.3.8.

      - Libraries: Chart.js for data visualization.

      - Data Source: Open-Meteo Historical Forecast API.
